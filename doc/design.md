# OpenStad Figma designs

While various Figma designs have been created over the years, none are a complete overview of all of OpenStad's features. Currently, there are two relevant design files, one of which is actively maintained.

## Admin panel designs

The current iteration of the admin panel is maintained code-first. When larger redesigns or more complex systems are to be made, a design is added to the OpenStad Admin Panel Figma design. This file is maintained by Draad. You can view this file [here](https://figmashort.link/W8dm4n). This file is actively maintained, in the sense that for new features, designs are created for them in this file.

Please note that, due to how this file is used, not all screens of the admin panel are represented in this design. The Figma file structure is also subject to change based on community feedback.

## Widget designs

Currently, there is no widget design file with a generic, non-specific style. Designs have been made internally for several organisations using their style guides. Please contact the OpenStad team if you would like to use these as a basis for your own organisation.

Please note that this Figma design is not actively maintained. Therefore, expect missing features that were added later, or unexpected layouts compared to the current OpenStad iteration. Nevertheless, it is the most up-to-date widget design collection at this moment.
